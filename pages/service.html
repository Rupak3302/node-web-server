Node.js Server
Capabilities

Everything this server can do — all powered by Node's built-in modules and zero external dependencies.

🗺️
URL Routing
Maps each pathname to a file. The route map is a plain object — easy to read, easy to extend. Unmatched routes fall through to 404.
Built-in
📂
Async File Serving
Uses fs.readFile with a callback — never blocks the event loop while reading HTML from disk.
Non-blocking
🔢
HTTP Status Codes
Every response carries the correct code: 200 for found pages, 404 for missing routes, 500 for server errors.
RFC Compliant
🛡️
Error Handling
Custom 404 page for unknown routes. Graceful 500 fallback if a file can't be read. Shutdown handled via SIGINT.
Resilient
